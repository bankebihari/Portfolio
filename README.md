# Portfolio

A personal developer portfolio built with React + Vite.

## 🌐 Live Demo

> **https://bankebihari-portfolio.vercel.app**

## Features

- Dark glassmorphism theme
- Auth-gated editing (ID + password protected)
- Editable hero name, skills, experience, projects
- Drag-to-reorder project cards
- Resume upload, download & delete
- Contact form (opens mail client)
- Welcome toast on visit
- Footer about chips (add/remove)
- Fully responsive (mobile + desktop)

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npx vercel --prod
```

## Tech Stack

- React 19
- React Router v7
- Vite
- CSS (glassmorphism dark theme)
- Vercel (deployment)
